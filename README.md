# Prayer Timer 🕌

A simple web app that displays daily Islamic prayer times for major cities in Saudi Arabia.

## Features
- Shows 6 prayer times: Fajr, Sunrise, Dhuhr, Asr, Maghrib, Isha
- Displays the current Hijri date in Arabic
- City selector for Makkah, Madinah, Eastern Province, and Riyadh
- Responsive design for mobile screens
- RTL Arabic interface

## Built With
- **HTML5 / CSS3** — structure and styling with Flexbox and Media Queries
- **JavaScript (ES6+)** — DOM manipulation and event handling
- **[Axios](https://axios-http.com/)** — HTTP client for API requests
- **[Aladhan API](https://aladhan.com/prayer-times-api)** — prayer times data provider
- **[Google Fonts](https://fonts.google.com/)** — Tajawal, Changa, El Messiri
